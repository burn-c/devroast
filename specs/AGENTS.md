# Specs

## Criar especificação antes de implementar

Para cada feature, criar um arquivo `.md` nesta pasta com:

```
# [Feature Name]

## Goal
Breve descrição do objetivo.

## Requirements
- Requisito 1
- Requisito 2

## Design
Referências visuais ou descrições.

## Technical
Stack/tecnologias específicas, se houver.
```

## Fluxo

1. Criar spec em `/specs/feature-name.md`
2. Revisar com o time
3. Aprovar antes de implementar
